# 配列とは？

- ロケット鉛筆のようなもの
- ロケット鉛筆でいう本体部分を、コンテナという
- ロケット鉛筆でいう芯の部分を、要素という

## 配列を操作する
### 1. 配列を作る
2つのやり方がある。よく取られる方法は前者。
a = []
b = list()

### 2. 配列の中身を確認する
c = [234, 739, 1902]

c[0]  # => 234
c[1]  # => 739

### 3. 配列から要素を追加・削除する
c.append(400)
c # => [234, 739, 1902, 400]

### 4. 配列同士をくっつける
d = [100, 200]
c.extend(d) # => [234, 739, 1902, 400, 100, 200]
または
c += d
